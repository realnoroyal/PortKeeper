Название: PortKeeper

Описание: Простая консольная утилита, предназначенная для сетевого тестирования и отладки конфигураций брандмауэров/роутеров.
Программа позволяет «занять» любой указанный сетевой порт в операционной системе Windows, предотвращая его закрытие со стороны ОС и симулируя работу активной сетевой службы.

Основные возможности:
    Динамический выбор: запрашивает номер порта прямо при старте (в диапазоне от 1 до 65535).
    Мультипротокольность: поддерживает работу с протоколами TCP, UDP, а также одновременный запуск обоих протоколов на одном порту с помощью многопоточности (threading).
    Логирование в реальном времени: отображает IP-адреса и порты всех внешних устройств, которые пытаются подключиться к вашему ПК или присылают пакеты данных.
